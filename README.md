# Customer-behavior-analysis
Data analytics project showcasing customer behavior analysis using  Python, SQL and PowerBI
# Overview
This project demonstrates a complete end-to-end data analytics workflow, starting from raw data processing to business insights. The project includes data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, dashboard creation, report generation, and presentation development.

The objective is to transform raw data into meaningful insights that support data-driven decision-making.

# Dataset
- Dataset Format: CSV
- Source: Customer Behavior Dataset
- Records: Contains customer-related information used for analysis.
- Features include demographic details, purchasing behavior, and other relevant attributes.

# Project Workflow

# 1. Data Loading
- Imported the dataset into Python.
- Loaded the data using Pandas.
- Verified data types and structure.

# 2. Exploratory Data Analysis (EDA)
- Examined dataset dimensions.
- Checked summary statistics.
- Identified missing values.
- Explored distributions.
- Created visualizations to understand patterns and relationships.

# 3. Data Cleaning
- Removed duplicate records.
- Handled missing values.
- Corrected inconsistent data.
- Converted data types where required.
- Prepared the dataset for analysis.

# 4. SQL Analysis
- Imported the cleaned dataset into a MySQL database.
- Wrote SQL queries to:
  - Filter records
  - Aggregate data
  - Group customer information
  - Calculate key metrics
  - Extract business insights

# 5. Power BI Dashboard
Created an interactive dashboard featuring:
- KPI Cards
- Customer Overview
- Sales/Purchase Analysis
- Demographic Analysis
- Interactive Filters
- Charts and Visualizations

# 6. Report Creation
Prepared a detailed project report covering:
- Problem Statement
- Methodology
- Data Cleaning
- EDA
- SQL Analysis
- Dashboard Insights
- Findings
- Recommendations

# 7. Presentation
Designed a professional presentation using Gamma AI summarizing:
- Project Overview
- Methodology
- Analysis
- Dashboard
- Key Findings
- Business Recommendations
